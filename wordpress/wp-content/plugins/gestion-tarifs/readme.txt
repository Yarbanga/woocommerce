== Description ==

<strong>Gestion de tarifs</strong> permet d'ajouter une liste de tarifs avec plusieurs param&egrave;tres et de les ins&eacute;rer dans les articles et les pages via un shortcode. l'extension ne b&eacute;n&eacute;ficie pas encore d'un syst&egrave;me de traduction.

== Installation ==

1. Recopier le contenu de l'archive dans le r&eacute;pertoire des extensions (wp-content/plugins) et activer dans le panneau des extensions du backoffice de Wordpress.
2. Ajouter, modifier ou supprimer les tarifs ainsi que les donn&eacute;es connexes puis utilisez les shortcodes associ&eacute;s pour inclure les tarifs ([tarif id="5"] par exemple) voire les intitul&eacute;s des tarifications ([intitule id="2"] par exemple).
